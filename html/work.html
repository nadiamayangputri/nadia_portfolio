About me

Foodtopia

Foodtopia is a website I created for one of my subject at University. It aims to increase children awareness on fresh food ingredients, by providing children to a dictionary of fresh food ingredients, and allowing them to create food journal.

This project was created as a team, and my role was to design and build the front-end portion of the website. I was responsible to come up with the design and lead the implementation of the front-end. I used tools such as moqups, canva and photoshop to design the elements of the page.

Sleigh group

This site is a brochure website for Sleigh group, created during my internship with the company. I developed the website using HTML, CSS, and JavaScript. And I designed the website using an online mock up tool, Moqups.

The design of the website uses minimalistic colours such as dark grey, blue and white. To add a little uniqueness to the simplistic site, I designed the layout to have slanted sections.

NaviGuide

This was my first Javascript self initiated project, it's a small project that helped me get a grip at programming using Javascript and Javascipt library, jQuery.

This website lets the user build a custom ice cream, hence its name. There are 4 parts of the ice cream. The container, ice cream flavor, sauce, and toppings. I used photoshop to create all the components of the ice cream. My aim was to keep the components simple yet visually pleasing.

The biggest challenge for me was to render the components in the correct position in the preview. However overall, this project was a lot of fun to make.
